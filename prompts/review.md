# Review Prompt

Review the recent changes against the current task.

Before reviewing:

1. Read `AGENTS.md`.
2. Read `spec.md`.
3. Read `tasks/current.md` or the task that was implemented.
4. Read `knowledge/docs-retrieval.md`.

Review checklist:

- [ ] Does the implementation satisfy acceptance criteria?
- [ ] Were correct docs used for external dependencies?
- [ ] Did it introduce unnecessary complexity?
- [ ] Did it break existing behavior?
- [ ] Are tests updated or run?
- [ ] Are docs updated if behavior changed?
- [ ] Are remaining risks documented?
- [ ] Is the code consistent with existing conventions?
- [ ] Were external API calls verified against Context or official docs?

Output:

## What changed?

## Does it match the task?

## Acceptance criteria result

## Docs consulted

## External API / library correctness

## Tests run

## Risks

## Unnecessary complexity found

## Docs updated?

## Recommended next task
